Important announcement: RESOURCES RELOCATED!

The URL of the SmartCampus repositories has changed to: ​https://github.com/smartcampuslab

Important announcement: SVN REPOSITORIES RELOCATED!

The URL of the SmartCampus repositories has changed to: ​http://svn.smartcampuslab.it/svn/ (previous was ​http://svn.smartcampuslab.it) If you use Eclipse you can right-click on the project you want to update, choose ‘relocate’ and insert the new URL. You can also relocate from command line using the following command: 
svn switch --relocate http://svn.smartcampuslab.it/repository/path http://svn.smartcampuslab.it/svn/repository/path

Important announcement: APP NEW NAMES!

SmartCampus services have been recently renamed. This change has been applied only to the Graphical User Interface of the services. The names of the Subversion repositories and other development and infrastructural resources have maintained the old names.
